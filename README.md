# 🖥️ Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/kauasilvandrade/teste-frontend-developer.git
```

### 2. Configurar servidor local
- Instale o **XAMPP**
- Start Apache e MySQL
- Coloque a pasta do projeto dentro da pasta:
  ```bash
  htdocs/
  ```

### 3. Criar o banco de dados
- Acesse o **phpMyAdmin** 
- http://localhost/phpmyadmin
- Crie um banco de dados chamado:
  
  ``` bash
      CREATE DATABASE contatoviagem;
  ```

- Crie a tabela contatos:
  
  ```bash
  CREATE TABLE contatos (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100),
    telefone VARCHAR(20),
    destino VARCHAR(100),
    mensagem TEXT
  );
  ```

### 4. Abrir no navegador
  - Abra o navegador e acesse:
    
    ``` bash
    http://localhost/teste-frontend-developer/
    ```
